## 📥 第一步：下载插件

目前插件为 **v1.2 内测版**，请通过以下官方链接下载安装包：

👉 **[点击下载最新版插件 (v1.2.zip)](https://github.com/Zephyr886/paper-trading/archive/refs/tags/v1.2.zip)**

> **⚠️ 注意：** 下载完成后，请务必将 `.zip` 压缩包 **解压** 到一个文件夹中（记住这个文件夹的位置）。

---

## 🛠 第二步：安装到浏览器

本插件支持 Chrome、Edge、Brave 等基于 Chromium 内核的浏览器。由于尚未上架应用商店，需手动安装：

1.  **打开扩展程序页面**：
    * 在浏览器地址栏输入：`chrome://extensions/` 并回车。
    * 或者：点击浏览器右上角的🧩拼图图标 -> “管理扩展程序”。

2.  **开启开发者模式**：
    * 找到页面右上角的 **“开发者模式” (Developer mode)** 开关，将其打开 🔵。

3.  **加载插件**：
    * 点击左上角出现的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
    * 选择你在第一步中 **解压出来的文件夹**。

🎉 **成功！** 你会在浏览器扩展栏看到“纸上谈币”的红色印章 Logo。

---

## 📈 第三步：开始模拟交易

1.  **进入战场**：
    打开 **GMGN.ai** (或其他支持的看线网站)。插件会自动识别并激活。

2.  **开启面板**：
    在页面上寻找悬浮的“纸上谈币”面板。

3.  **买入与卖出**：
    * 输入模拟金额（例如 1 SOL）。
    * 点击 **[买入]** 或 **[卖出]**。
    * 所有盈亏数据均为本地模拟，**不消耗真实 Gas，不涉及链上签名**，绝对安全。

## ❓ 常见问题

**Q: 安装时提示“未找到清单文件”？**
A: 请确认你加载的是 **解压后的文件夹**，而不是 `.zip` 压缩包本身。

**Q: 插件是免费的吗？**
A: 完全免费开源。本项目旨在构建新人友好的 Meme 教学社群。

**Q: 为什么官网没有自动播放音乐？**
A: 浏览器限制自动播放。你需要点击一下页面（或点击右下角的🎵按钮）来“解锁”音频。
